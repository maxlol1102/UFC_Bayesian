# UFC_Bayesian
A machine learning model that predicts fight outcomes in the Ultimate Fighting Championship (UFC) using historical fight data and fighter statistics.

UFC Fight Outcome Prediction Model

This project focuses on predicting fight outcomes in the Ultimate Fighting Championship (UFC) using machine learning techniques. The goal is to analyze historical fight data and fighter statistics to build a predictive model that estimates the probability of a fighter winning a match.
